# Browser Reading Plugin

一个轻量 Chrome / Edge 阅读增强插件，用来改善在线小说站点的阅读体验。它不会提取或重排正文结构，而是对当前页面应用更协调的字体、背景、宽度、行高和轻度广告浮层净化。

## 功能

- 当前站点一键启用或停用阅读增强。
- 主题切换：纸张、护眼、亮色、暖色、暗色、纯黑。
- 字体切换：衬线、无衬线、楷体、系统、等宽。
- 调整字号、行高、段落间距、正文宽度和页面留白。
- 可选弱化广告、浮层、侧栏、推荐区等干扰元素。
- 设置按站点域名保存，不同小说站可以使用不同阅读配置。

## 安装

1. 打开 Chrome 或 Edge。
2. 进入扩展管理页：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
3. 打开“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择本项目目录：`C:\Users\Administrator\Documents\Browser Reading Plugin`。

## 使用

1. 打开一个在线小说章节页面。
2. 点击浏览器工具栏里的“阅读增强”扩展按钮。
3. 打开右上角开关。
4. 按需要调整主题、字体、字号、行高、段落间距、正文宽度和页面留白。

如果扩展刚安装后对已经打开的页面无效，刷新小说页面后再打开插件即可。

## 兼容性说明

第一版采用“样式增强 + 轻度净化”方式，适配范围比强制正文提取更广，也更不容易破坏章节翻页、目录按钮和站点原有功能。部分站点如果把正文放在特殊 iframe 或 Shadow DOM 中，可能无法完全调整。
